# Subnets

The VPC is divided into multiple subnets.

## Public Subnets

- Bastion Host
- Application Load Balancer

## Private Application Subnets

- EC2 App Server 1
- EC2 App Server 2

## Private Database Subnets

Reserved for Amazon RDS (not deployed due to AWS Free Plan limitations).

## Benefits

- High Availability
- Improved Security
- Better Network Segmentation
